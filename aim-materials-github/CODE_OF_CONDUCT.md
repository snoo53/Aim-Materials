# Code of Conduct

## Our Standards

We expect contributors to:

- be respectful and professional,
- focus on technical issues and evidence,
- provide constructive feedback,
- avoid harassment, discrimination, and personal attacks.

## Unacceptable Behavior

- Harassment, insults, or abusive language
- Discriminatory or exclusionary comments
- Publishing private information without consent
- Any conduct that is inappropriate in a professional open-source project

## Enforcement

Project maintainers may remove or edit comments, issues, commits, or pull requests that violate this code, and may restrict participation for repeated or severe violations.

## Reporting

Please report concerns through a private GitHub channel (security advisory or direct maintainer contact in `SECURITY.md`).
